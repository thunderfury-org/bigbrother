# BigBrother

A tool for managing and organizing TV show files and metadata.

## Features

- Smart filename parsing for TV shows
- Integration with TMDB for show information
- AList support for cloud storage management
- Automated episode filename renaming

## Configuration

See `config.template.yaml`.

Make sure to set up your TMDB API key and AList credentials before using the application.
